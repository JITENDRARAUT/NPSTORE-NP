# 🛒 Np Store

A modern, clean, and fully responsive e-commerce web application storefront. This project focuses on building an intuitive user interface for online shopping, complete with structured navigation and user authentication templates.


-----

## ✨ Features

*   🛍️ **Interactive Storefront:** Clean layout designed for viewing products seamlessly across desktop and mobile devices.
*   🔐 **User Authentication Pages:** Pre-built interfaces for both User Sign-Up (`signup.html`) and Secure Login (`login.html`).
*   👤 **User Dashboard:** Dedicated profile management view (`profile.html`) for handling user account details.
*   📱 **Responsive Design:** Fully optimized layout using semantic HTML5 and modern CSS techniques.

---

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
*   **Deployment:** GitHub Pages
